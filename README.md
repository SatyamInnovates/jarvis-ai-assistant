JARVIS-AI-ASSISTANT

jarvis is a python based voice assistant and a chatbot inspired by Iron man ai assistant.
when you run this , it asks to choose mods like text nad voice.
it uses asyncio so the code is optimized and it also keeps record of the chats.


Technology used:
-asyncio
-python
-time
-tts
-google.genai
-sr



HOW to use

install python(3.9 or above)
install required libraries:
-re
-asyncio
-pygame
-time
-google
-google.generativeai
-speech_recognition
-datetime

2nd step
add your service_account_path that you will get from google.cloud
add your gemini_api_key

3rd step :
run and use
